# WebAssembly AI Emotion Processing

This WebAssembly module powers AI-driven emotion analysis to adapt UI based on user sentiment.

## Build Instructions

1. Install Rust and WebAssembly target:
   ```bash
   rustup target add wasm32-unknown-unknown
   ```

2. Build the WebAssembly module:
   ```bash
   cargo build --target wasm32-unknown-unknown --release
   ```

3. Integrate with frontend applications.
